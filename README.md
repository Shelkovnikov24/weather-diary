## Описание программы
**Expense Tracker** — это графическое приложение (GUI) для учета личных финансов. Программа позволяет фиксировать траты по категориям, автоматически подсчитывать сумму за период и фильтровать записи. Данные надежно сохраняются в локальную базу данных (формат JSON).

## Функциональные возможности
- **Добавление расходов:** ввод суммы, выбор категории (Еда, Транспорт и т.д.) и даты.
- **Сохранение и загрузка:** автоматическая работа с файлом `expenses.json`.
- **Валидация (защита от ошибок):** строгая проверка на ввод положительного числа и корректный формат даты (ГГГГ-ММ-ДД).
- **Аналитика:** фильтрация записей по категориям и динамический подсчет итоговой суммы.
