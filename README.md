## Описание программы
**Weather Diary** — это десктопное GUI-приложение для ведения личного дневника погоды. Программа позволяет сохранять ежедневные метеосводки, фиксируя температуру, наличие осадков и текстовое описание. Данные сохраняются локально в формате JSON. Реализована система фильтрации для быстрого поиска записей по дате или температурному режиму.

## Функциональные возможности
- **Добавление записей:** ввод даты, температуры, наличия осадков и текстового описания.
- **Сохранение и загрузка:** автоматическая работа с файлом `weather.json`.
- **Валидация:** проверка формата даты (ГГГГ-ММ-ДД), проверка температуры (только числа), запрет на пустое описание.
- **Фильтрация:** - Поиск записей за конкретный день.
  - Отображение дней, где температура была выше заданного значения (например, выше +10°C).
